## YOUR NEXT STEPS (Workflow)

1.  **Download**: Search and download the datasets above (they will be `.zip` files).
2.  **Extract**: Unzip them.
3.  **Organize**: Create folders in your project exactly like this:
    ```
    c:/Project/CDD/LeafModel/data/raw/Mandua/Blast/
    c:/Project/CDD/LeafModel/data/raw/Arhar/SterilityMosaic/
    c:/Project/CDD/LeafModel/data/raw/Turmeric/LeafBlotch/
    ```
    *(Put the images inside the corresponding disease folders)*

4.  **Run**:
    *   If using **Colab**: Upload the whole `LeafModel` folder to Drive, then open `Run_in_Colab.ipynb`.
    *   If avoiding Colab: Just run `python src/train.py` on your PC.
